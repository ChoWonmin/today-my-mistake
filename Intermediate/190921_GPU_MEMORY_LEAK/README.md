# GPU Memory Leak in Vue

> Browser에서 GPU 접근하는 작업은 메모리 누수에 주의해야한다. 특히 라우팅 변경을 할 때, 현재 연결된 GPU 메모리를 끊어주지 않으면 다음 라우팅에서도 그대로 남아서 메모리 누수가 발생한다.
